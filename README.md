# Docker
- sudo docker build -t my-nodejs-app .
- sudo docker run -it --rm  -v path/to/code4moldova.github.io:/homeapp -p 8882:8882 --name my-running-app my-nodejs-app
- `nodemon server.js`
- go to localhost:8882
